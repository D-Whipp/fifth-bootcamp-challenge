# Work Day Scheduler

### Front End Web App

![app screenshot](./assets/images/app-screenshot.png)

## Description

Work Day Scheduler is a very basic app that helps you keep track of your day's schedule.

## Table of Contents

## Usage

To use work day schedule click [here](https://d-whipp.github.io/work-day-scheduler/) and you'll be taken to the website. Once there you can enter information you'd like to keep track of for the day. <br>
This app still needs some work, namely to persist data past a page refresh.
